# TclTacToe
Tic Tac Toe in Tcl

How to play on Debian Linux:

- `sudo apt-get update`
- `sudo apt-get install tcl`
- navigate to project folder
- run `tclsh tictactoe.tcl`


How to play on Fedora Linux:

- `sudo dnf update`
- `sudo dnf install tcl`
- navigate to project folder
- run `tclsh tictactoe.tcl`


How to play on Arch Linux:

- `sudo pacman -Syu`
- `sudo pacman -S tcl`
- navigate to project folder
- run `tclsh tictactoe.tcl`
